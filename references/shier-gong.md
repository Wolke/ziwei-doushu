# 十二宮位詳解

## 宮位概述

紫微斗數以十二宮位為基本框架，每個宮位代表人生的不同面向。十二宮位以命宮為起點，逆時針排列。

---

## 命宮

### 基本含義

命宮是整個命盤的核心，代表命主本人的一切：

- **個性特質**：天生的性格傾向
- **人生格局**：一生的基本格局高低
- **行為模式**：處事風格與習慣
- **外貌氣質**：外在形象與氣場

### 分析重點

1. 命宮主星決定基本性格
2. 命宮輔煞星影響格局高低
3. 命宮四化決定人生起伏
4. 三方四正星曜影響整體運勢

### 各主星在命宮

| 主星 | 性格特點 |
|------|----------|
| 紫微 | 領導氣質、高傲、有主見 |
| 天機 | 聰明機變、善謀略、好動 |
| 太陽 | 光明磊落、熱心、愛面子 |
| 武曲 | 剛毅果決、重財、較孤僻 |
| 天同 | 溫和隨和、享樂、較懶散 |
| 廉貞 | 精明複雜、愛恨分明 |
| 天府 | 穩重保守、重名譽 |
| 太陰 | 情緒敏感、內斂、愛美 |
| 貪狼 | 多才多藝、慾望強 |
| 巨門 | 口才好、多疑、是非多 |
| 天相 | 隨和輔佐、守規矩 |
| 天梁 | 正直清高、喜助人 |
| 七殺 | 剛強衝勁、獨立 |
| 破軍 | 開創變動、不安定 |

## 兄弟宮

### 基本含義

- **兄弟姐妹**：手足關係
- **同輩朋友**：平輩的人際關係
- **合作夥伴**：同事、合夥人
- **母親身體**：部分派別認為

### 分析重點

1. 兄弟數量與緣分
2. 手足之間的關係
3. 是否能得到手足幫助
4. 與同事的相處模式

### 吉凶判斷

| 情況 | 意義 |
|------|------|
| 吉星多 | 手足和睦、互相幫助 |
| 煞星多 | 手足不和、競爭衝突 |
| 化忌入 | 手足有事端 |
| 空宮 | 手足緣薄或獨生 |

## 宮位關係

### 三方四正

每個宮位的三方四正：

| 本宮 | 對宮 | 三合宮 |
|------|------|--------|
| 命宮 | 遷移宮 | 財帛宮、事業宮 |
| 兄弟宮 | 交友宮 | 疾厄宮、田宅宮 |
| 夫妻宮 | 事業宮 | 遷移宮、福德宮 |
| 子女宮 | 田宅宮 | 財帛宮、父母宮 |
| 財帛宮 | 福德宮 | 命宮、事業宮 |
| 疾厄宮 | 父母宮 | 兄弟宮、交友宮 |

### 宮位互化

- **對宮**：相互影響、互為表裡
- **三合宮**：力量相加、互相支援
- **夾宮**：夾制作用、影響本宮
