# 煞星詳解

## 煞星概述

煞星又稱「六煞星」，對命盤有負面影響，但有時也能激發潛能。煞星的作用需結合整體命盤分析。

---

## 擎羊星

### 基本資料

- **五行**：庚金（陽金）
- **別稱**：羊刃、刑星
- **主事**：衝突、傷害

### 特性

- 剛烈直接
- 容易衝突
- 開刀傷害
- 競爭心強

### 正面作用

- 有衝勁
- 執行力強
- 不怕困難
- 有開創力

### 負面作用

- 衝動魯莽
- 容易受傷
- 與人衝突
- 官非是非

### 各宮意義

| 宮位 | 意義 |
|------|------|
| 命宮 | 性格剛烈 |
| 疾厄宮 | 手術、傷害 |
| 事業宮 | 職場衝突 |
| 夫妻宮 | 婚姻不順 |
| 遷移宮 | 出外有災 |

### 位置規則

擎羊在祿存前一宮（順時針方向）

- 意外傷害

### 各宮意義

| 宮位 | 意義 |
|------|------|
| 命宮 | 性格急躁 |
| 疾厄宮 | 急性病 |
| 事業宮 | 工作不穩 |
| 夫妻宮 | 感情急變 |

- 實業不宜

### 正面影響

- 創意工作

- 從事精神層面工作

- 修身養性
- 布施行善

## 煞星組合

### 羊陀夾命

擎羊、陀羅夾命宮兩側：
- 困難重重
- 進退兩難
- 格局降低

### 火鈴夾命

火星、鈴星夾命宮兩側：
- 意外災禍
- 急躁暗傷
- 需要謹慎

### 空劫夾命

地空、地劫夾命宮兩側：
- 財運不佳
- 虛耗損失
- 宜從事藝術

### 六煞入命

多顆煞星入命宮：
- 格局降低
- 但有衝勁
- 需要化解

---

## 煞星化解

### 化解方法

1. **吉星解煞**：輔星可化解煞星
2. **行運轉化**：等待好運
3. **職業選擇**：從事適合的行業
4. **修身養性**：提升修養

### 煞星不一定凶

- 身強命旺者可駕馭煞星
- 煞星可激發潛能
- 需看整體命盤配置
