# 四化飛星詳解

## 四化概述

四化是紫微斗數中最重要的動態元素，由年干決定，代表星曜的能量轉化。

| 四化 | 性質 | 能量 | 作用 |
|------|------|------|------|
| **化祿** | 吉 | 增益 | 財富、順利、人緣 |
| **化權** | 吉 | 掌控 | 權力、主導、競爭 |
| **化科** | 吉 | 名聲 | 貴人、考試、名望 |
| **化忌** | 凶 | 阻礙 | 執著、是非、困難 |

| 天同化祿 | 福氣增加、享受 |

| 貪狼化祿 | 桃花財、偏財運 |

## 四化組合

### 三吉化

當化祿、化權、化科同時出現在三方四正：

- **三奇嘉會**：名利雙收、大吉
- **雙祿交流**：祿存與化祿同宮，財運極佳

### 化忌組合

| 組合 | 意義 |
|------|------|
| 雙忌同宮 | 困難加倍 |
| 羊陀夾忌 | 極為不利 |
| 火鈴夾忌 | 意外災禍 |
| 忌沖命宮 | 衝擊較大 |

## 飛星四化

### 本命四化

由出生年干決定，影響一生。

### 大限四化

由大限天干決定，影響該十年。

### 流年四化

由流年天干決定，影響該年。

### 四化重疊

- **疊祿**：雙重好運
- **疊權**：權力極大
- **疊科**：名聲極佳
- **疊忌**：問題嚴重

---

## 化忌解法

### 化解方式

1. **化科解忌**：科星可緩解忌星
2. **祿存解忌**：祿存可減輕化忌
3. **行運轉化**：等待好運到來
4. **外應輔助**：佩戴、布局等

### 注意事項

- 化忌不一定全凶，有時代表執著、專注
- 要看化忌所在宮位和星曜
- 結合整體命盤分析
