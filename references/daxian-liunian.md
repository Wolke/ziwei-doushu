# 大限流年推算

## 大限概述

大限是紫微斗數中十年一運的運勢週期，代表人生各階段的運勢走向。

---

## 大限計算

### 起運規則

大限的起始年齡由五行局決定：

| 五行局 | 起運年齡 |
|--------|----------|
| 水二局 | 2歲 |
| 木三局 | 3歲 |
| 金四局 | 4歲 |
| 土五局 | 5歲 |
| 火六局 | 6歲 |

### 順逆規則

| 條件 | 排法 |
|------|------|
| 陽年男 | 順行 |
| 陽年女 | 逆行 |
| 陰年男 | 逆行 |
| 陰年女 | 順行 |

**陽年**：甲、丙、戊、庚、壬年
**陰年**：乙、丁、己、辛、癸年

### 大限宮位

從命宮開始，每十年走一個宮位：

| 大限 | 年齡段（以土五局為例） |
|------|------------------------|
| 第一大限 | 5-14歲 |
| 第二大限 | 15-24歲 |
| 第三大限 | 25-34歲 |
| 第四大限 | 35-44歲 |
| 第五大限 | 45-54歲 |
| 第六大限 | 55-64歲 |
| 第七大限 | 65-74歲 |
| 第八大限 | 75-84歲 |

## 大限分析

### 看大限宮位

1. **大限命宮**：大限走到的宮位
2. **大限主星**：該宮位的主星
3. **大限輔煞**：該宮位的輔星和煞星
4. **大限四化**：大限天干產生的四化

### 大限天干

大限宮位的天干產生新的四化：

| 大限宮位地支 | 大限天干 |
|-------------|----------|
| 子、丑 | 根據宮位的天干 |
| 寅、卯 | ... |
| ... | ... |

### 大限三方四正

分析大限命宮的三方四正：
- 大限財帛宮
- 大限事業宮
- 大限遷移宮

## 流年推算

### 流年命宮

流年命宮 = 太歲所在宮位

例如：
- 子年：流年命宮在子宮
- 丑年：流年命宮在丑宮
- ...

### 流年十二宮

從流年命宮開始，逆時針排列十二宮：
- 流年命宮
- 流年兄弟宮
- 流年夫妻宮
- ...

### 流年四化

由流年天干決定四化：

| 流年 | 天干 | 四化 |
|------|------|------|
| 2024 | 甲 | 廉貞化祿、破軍化權... |
| 2025 | 乙 | 天機化祿、天梁化權... |
| 2026 | 丙 | 天同化祿、天機化權... |
| ... | ... | ... |

## 小限推算

### 小限定義

小限是紫微斗數中每年一宮的運勢週期。

### 小限起點

從命宮開始，每年走一宮：
- 男命順行
- 女命逆行

### 小限與流年

小限與流年都代表當年運勢，需結合分析。

---

## 流月流日

### 流月

流月 = 流年的月份運勢
- 正月：流月命宮在流年命宮
- 依次排列...

### 流日

流日 = 當日的運勢
- 初一：流日命宮在流月命宮
- 依次排列...
