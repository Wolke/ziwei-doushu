# 輔星詳解

## 輔星概述

輔星又稱「六吉星」，對命盤有正面輔助作用，能提升格局、增加貴人運。

---

## 左輔星

### 基本資料

- **五行**：戊土
- **別稱**：助星
- **主事**：輔佐、助力

### 特性

- 代表男性貴人
- 輔佐能力強
- 增加人緣
- 提升格局

### 各宮意義

| 宮位 | 意義 |
|------|------|
| 命宮 | 有貴人相助 |
| 事業宮 | 工作有助手 |
| 夫妻宮 | 感情有幫助 |
| 財帛宮 | 財運有助力 |
| 遷移宮 | 外出有貴人 |

### 與主星配合

- 與紫微同宮：權威加強
- 與天府同宮：穩重有助
- 單獨入命：有貴人但需靠自己

## 文昌星

### 基本資料

- **五行**：庚金
- **別稱**：文星
- **主事**：學業、文書

### 特性

- 代表正統學問
- 考試運佳
- 文書能力
- 思維清晰

### 各宮意義

| 宮位 | 意義 |
|------|------|
| 命宮 | 聰明好學 |
| 事業宮 | 適合文職 |
| 父母宮 | 學業順利 |
| 遷移宮 | 出外有名 |
| 財帛宮 | 靠知識生財 |

### 注意事項

- 文昌化忌：文書錯誤、考試不順
- 與桃花星同宮：才子佳人
