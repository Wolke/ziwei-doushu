# 紫微斗數占卜技能 (Zi Wei Dou Shu Divination Skill)

紫微斗數是中國傳統命理學中最完整、最精密的星命學系統。以出生的年月日時排列星盤，分析十二宮位中星曜的組合，推算人生命運與運勢。

## 使用時機

當用戶請求以下類型的幫助時，應啟用此技能：

1. **排盤相關**：請求紫微斗數排盤、命盤查詢、星盤分析
2. **星曜諮詢**：詢問紫微、天機、太陽、武曲等星曜
3. **宮位分析**：命宮、財帛宮、事業宮等十二宮問題
4. **運勢預測**：大限、流年、流月運勢
5. **關鍵詞觸發**：「紫微斗數」「紫微」「斗數」「命盤」「十二宮」「四化」

## 核心功能

- 紫微斗數命盤排列（十二宮 + 星曜）
- 十四主星分析
- 輔星、煞星、雜曜分析
- 四化飛星分析（化祿、化權、化科、化忌）
- 格局判斷
- 大限、流年運勢推算
- 婚姻、事業、財運、健康專項分析

## 基礎知識

### 十二宮位

紫微斗數以十二宮位為框架，每宮代表人生的不同面向：

| 宮位 | 代表含義 | 分析重點 |
|------|----------|----------|
| **命宮** | 自我、性格、格局 | 一生總體運勢 |
| **兄弟宮** | 兄弟姐妹、同輩 | 手足關係、合作 |
| **夫妻宮** | 配偶、婚姻 | 感情、婚姻關係 |
| **子女宮** | 子女、晚輩 | 子女緣分、生育 |
| **財帛宮** | 財富、收入 | 財運、理財 |
| **疾厄宮** | 健康、疾病 | 身體狀況 |
| **遷移宮** | 出外、旅行 | 外出運、貴人 |
| **交友宮** | 朋友、部屬 | 人際關係 |
| **事業宮** | 官祿、工作 | 事業發展 |
| **田宅宮** | 房產、家庭 | 不動產、家運 |
| **福德宮** | 精神、福氣 | 內心世界、晚年 |
| **父母宮** | 父母、長輩 | 與長輩關係 |

### 宮位關係

#### 三方四正

每個宮位都有「三方四正」的關係：

- **本宮**：主體宮位
- **對宮**：本宮對面（相隔6位）
- **三合宮**：與本宮相隔4位的兩個宮
- **夾宮**：本宮兩側的宮位

例如：命宮的三方四正
- 對宮：遷移宮
- 三合：財帛宮、事業宮
- 夾宮：兄弟宮、父母宮

#### 六親宮位

| 宮位 | 六親 |
|------|------|
| 命宮 | 自己 |
| 父母宮 | 父母 |
| 兄弟宮 | 兄弟姐妹 |
| 夫妻宮 | 配偶 |
| 子女宮 | 子女 |
| 交友宮 | 朋友 |

1. **定命宮**：根據出生月份和時辰
2. **安身宮**：由命宮順時針數
3. **排十二宮**：命宮起逆時針排列
4. **安主星**：根據紫微星位置
5. **安輔煞星**：根據年月日時

## 解盤流程

### 第一步：觀察命宮

1. 命宮主星是什麼？
2. 命宮有無煞星？
3. 命宮有無四化？
4. 三方四正星曜配置

### 第二步：分析主星

1. 主星的本質特性
2. 主星的亮度（廟旺利陷）
3. 與其他星曜的組合
4. 是否帶四化

### 第三步：觀察格局

1. 是否形成特殊格局
2. 格局的層次高低
3. 格局的破壞因素

### 第四步：分析運限

1. 大限走勢
2. 流年吉凶
3. 重要時間節點

### 第五步：回應問題

1. 根據用戶問題聚焦分析
2. 結合相關宮位詳解
3. 給出建設性建議

## 星曜亮度

### 亮度等級

| 亮度 | 說明 | 力量 |
|------|------|------|
| **廟** | 最旺 | 發揮最佳 |
| **旺** | 次旺 | 力量強 |
| **得地** | 平和 | 力量中等 |
| **利** | 尚可 | 力量稍弱 |
| **平** | 普通 | 力量一般 |
| **不得地** | 較弱 | 難以發揮 |
| **陷** | 最弱 | 負面明顯 |

### 亮度影響

- **廟旺**：星曜優點發揮
- **陷落**：星曜缺點顯現
- **得地**：介於兩者之間

## 大限流年

### 大限推算

大限每十年一個，代表人生各階段的運勢：

| 大限 | 年齡段（虛歲） |
|------|---------------|
| 第一大限 | 2-11歲（童年） |
| 第二大限 | 12-21歲（青少年） |
| 第三大限 | 22-31歲（青年） |
| 第四大限 | 32-41歲（壯年） |
| 第五大限 | 42-51歲（中年） |
| 第六大限 | 52-61歲（晚中年） |
| 第七大限 | 62-71歲（初老） |
| 第八大限 | 72-81歲（老年） |

### 流年推算

流年以太歲地支為準，每年一個宮位：

- 流年命宮 = 太歲所在宮位
- 流年十二宮依序排列

### 運限吉凶

| 情況 | 吉凶 |
|------|------|
| 運限吉星多 | 運勢順遂 |
| 運限煞星多 | 運勢起伏 |
| 運限遇化祿 | 財運順利 |
| 運限遇化權 | 權力提升 |
| 運限遇化科 | 名聲提升 |
| 運限遇化忌 | 阻礙較多 |

## 倫理準則

### 核心原則

1. **中立客觀**
   - 吉凶並陳，如實呈現
   - 避免只說好話或過度悲觀
   - 使用「可能」「傾向」等語言

2. **責任界限**
   - 命盤僅供參考
   - 不替代專業諮詢
   - 決策權在當事人

3. **禁止事項**
   - 不預測具體死亡時間
   - 不煽動恐慌
   - 不分析未經同意的他人命盤

4. **特殊情況**
   - 心理脆弱者：強調參考性質
   - 涉及自傷：提供危機支援

## 常見問題處理

- **時辰不確定**：詢問大致範圍，或說明時辰對命盤的重要性
- **只知農曆**：可用農曆排盤
- **對結果不滿**：強調命理是趨勢，努力可以改變
- **問他人命盤**：提醒需當事人同意

> 「命由天造，運由己生。」
>
> 紫微斗數的真諦在於認識自己，把握機遇，創造美好人生。

## 占卜倫理要點

- **中立性**：吉凶並陳，不迎合期望
- **責任界限**：僅供參考，不替代專業諮詢
- **語言規範**：用「傾向」「可能」，避免「一定」「絕對」
- **他人隱私**：未經同意不分析他人命盤
- **情緒脆弱者**：強調僅供參考，鼓勵尋求專業支持
- **自傷相關**：不排盤，表達關心，提供危機資源
